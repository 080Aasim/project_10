SCIENTIFIC CALCULATOR

RULE TO USE THIS CALCULATOR

Click the button first then the operation button that you want to do.

EXAMPLE :-

click on button "2" then if you want to perform log operation then click on "log";

BUTTONS THAT DOES NOT WORKS :-

  • (
  • )
  • +/-
  • RND
  • EXP
  • 1/x
  • sin -1
  • cos -1
  • tan -1
  • MR -1
